# GitHub Pages setup

This repository now includes a simple one-page website (index.html, styles.css, script.js).

To publish the site using GitHub Pages:

1. Go to the repository on GitHub: https://github.com/cvhandytech-ctrl/HandyTech/settings/pages
2. Under "Build and deployment" choose "Deploy from a branch".
3. For the branch choose: `main` and the folder: `/ (root)` (or choose `/docs` if you prefer placing files there).
4. Save. GitHub will give you a URL like `https://cvhandytech-ctrl.github.io/HandyTech/` after the site builds.

Notes:
- Replace the placeholder phone and email in `index.html` with your real contact details.
- If you prefer the site in `/docs`, move the three files into a `docs/` folder and update settings to serve from `/docs`.
- For a working contact form, sign up for a form provider (Formspree, Getform, Netlify Forms) and replace the form action.
